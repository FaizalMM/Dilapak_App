# Dilapak Mobile App

Aplikasi mobile pelayanan administrasi dan kependudukan Kota Madiun berbasis Flutter yang dirancang untuk mempermudah masyarakat dalam mengakses layanan publik secara digital melalui smartphone.

Dilapak Mobile App merupakan pengembangan versi mobile dari website layanan administrasi Kota Madiun dengan tampilan yang lebih modern, mobile-friendly, dan mudah digunakan. Aplikasi ini membantu masyarakat dalam melakukan pengajuan layanan kependudukan secara online tanpa harus datang langsung ke kantor pelayanan.

## Features

- Login & Registrasi Pengguna
- Verifikasi Akun
- Pengajuan Layanan Online
- Layanan Three In One
- Upload Dokumen Persyaratan
- Tracking Status Permohonan
- Riwayat Pengajuan
- Notifikasi Layanan
- Profil Pengguna
- Search Layanan
- Quick Action Menu

## Layanan Tersedia

### Kependudukan

- Penerbitan KTP-el
- KTP-el Hilang/Rusak
- Perubahan Data KTP
- KTP-el Pindah Datang
- KTP-el Luar Domisili
- KTP-el Ganti Foto/TTD
- SKPWNI / SKPWNA
- SKTT WNA

### Kartu Keluarga & KIA

- Penerbitan KK
- KK Hilang/Rusak
- Perubahan Data KK
- KIA WNI
- KIA WNA

### Pencatatan Sipil

- Akta Kelahiran
- Akta Kematian
- Akta Perkawinan
- Akta Perceraian
- Perubahan Nama
- Pengangkatan & Pengesahan Anak
- Keabsahan Akta

## UI/UX Concept

Aplikasi dirancang menggunakan konsep:

- Modern Government App
- Mobile First Design
- Clean & Simple Interface
- Fast Public Service Access
- User Friendly Navigation

Tampilan aplikasi tidak menyalin website secara langsung, tetapi dibuat ulang khusus untuk pengalaman pengguna mobile agar lebih nyaman, cepat, dan efisien.

## Installation

```bash
flutter pub get
flutter run
```
